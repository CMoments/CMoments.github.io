ToDo

To get information on how to configure To Do List, explore our documentation: https://docs.dhtmlx.com/todolist/how_to_start/.

You can import DHTMLX To Do List into your projects using yarn or npm package manager. To get the trial version of To Do List, run the following commands:

// yarn

yarn config set @dhx:registry https://npm.dhtmlx.com

yarn add @dhx/trial-todolist

Or

// npm

npm config set @dhx:registry https://npm.dhtmlx.com

npm i @dhx/trial-todolist
