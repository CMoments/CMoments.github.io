智能体设计模式

作者是谁？—— Antonio Gulli

- 现任：
Google Cloud 资深工程总监、杰出工程师（Distinguished Engineer）。
- 行业地位：
欧洲最早一批投身云原生与分布式系统的技术领袖，曾执掌微软 Azure 欧洲研发中心、AWS 欧洲实验室。
- 写作风格：
爱在 GitHub、Google Docs 上“边写边开源”，把 400 多页手稿免费扔给社区，再让出版社“慢半拍”收版税——这次版税全部捐给 Save the Children。
- 个人信条：
“If you can’t reuse it, you don’t understand it.” 于是把 20 年大规模系统经验拆成 21 张“图纸”，就是今天的主角——《Agentic Design Patterns》。

A Hands-On Guide to Building Intelligent Systems——“手把手教你搭智能系统”。全书 424 页，21 个核心模式 + 7 个高阶附录，用“模式语言”把 AI Agent 从玩具级 Demo 提升到工业级可维护系统。

软件时代我们有《Design Patterns》让代码可维护；
智能体时代我们有《Agentic Design Patterns》让 AI 可治理。
Antonio Gulli 把 20 年经验开源成 21 张图纸，剩下的就看你怎么搭出下一代 Killer Agent 了。
